## ЁЯФД Cycle Language Function

```javascript
/**
 * Cycle through languages: EN тЖТ MR тЖТ TE тЖТ TA тЖТ EN
 */
function cycleLanguage() {
  const languageOrder = ['en', 'mr', 'te', 'ta'];
  const currentLang = I18n.getCurrentLanguage();
  const currentIndex = languageOrder.indexOf(currentLang);
  const nextIndex = (currentIndex + 1) % languageOrder.length;
  const nextLang = languageOrder[nextIndex];
  
  changeLanguage(nextLang);
}
```

**Usage:**
```javascript
// Click a button
<button onclick="cycleLanguage()">ЁЯФД Cycle Language</button>

// Or programmatically
cycleLanguage(); // EN тЖТ MR
cycleLanguage(); // MR тЖТ TE
cycleLanguage(); // TE тЖТ TA
cycleLanguage(); // TA тЖТ EN
```
